To test website, there are two XML files in the /testdata directory.

Website has been tested on latest version of Google Chrome.

XML file must be in this format:
<?xml version="1.0" encoding="UTF-8"?>
<programmedata>
  <title>Programme Data</title>
  <programme id>
      <name>A nightmare on Elm Street</name>
      <image>assets/elm-street.jpg</image>
      <mood>Scared</mood>
  </programme>
</programmedata>
